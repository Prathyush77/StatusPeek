# StatusPeek

> Instantly view the HTTP status code of the current website.

## 🚀 Overview

StatusPeek is a Chrome Extension designed for Developers, DevOps Engineers, SREs, QA Engineers, and Operations teams.

Instead of opening DevTools just to check the response status, StatusPeek displays the HTTP status code instantly.

---

## ✨ Planned Features

- ✅ HTTP Status Code
- ⏳ Response Time
- ⏳ Redirect Chain
- ⏳ Server Header
- ⏳ Content-Type
- ⏳ TLS Version
- ⏳ Request History
- ⏳ Export Logs
